Nowadays society is very competitive and demanding, we need to work to earn money but also, but we need to study to get a job, and not only studies are necessary to have it, but also, they are looking for experience, courses, and languages, they are a lot of things but rarely we think about ourself, these things that the society thinks that we need to be Successful are making a lot of pressure and affect us.

We are talking about these topics because we are in the same situation, and this project will talk about society’s pressure and how it is affecting us. Being alone in these situations isn’t fair and we want to help other people that are having the same circumstance.

In this project you will see, the expectations that society has for us in a work and study context, and information about how we feel about this situation and how it is affecting us. Also explain the barriers that we have and finally a list of possible solutions that can help us.
